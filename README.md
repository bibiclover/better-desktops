Small program which calls `winvd` via `global-hotkey` instead of using an AutoHotKey script.

Adds useful shortcuts for Windows' Virtual Desktops.

---

Use Alt + 1,2,3,...,9,0 to move directly to the 1,2,3,...,9,10th desktop.

Use Ctrl + Alt + 1,2,3,...,9,0 to move the currently focused window to the 1,2,3,...,9,10th desktop.

Use Ctrl + Alt + ArrowLeft/ArrowRight to move the currently focused window to the left/right desktop.

Use Ctrl + Alt + P to pin/unpin the currently focused window.

Exit via the Tray Menu.
